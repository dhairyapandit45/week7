# My Frontend Deployment Folder

This is where I keep the code that is ready to be deployed to Vercel. 

## What's in here?
It's basically a copy of my main frontend folder but optimized for deployment.

## How to deploy:
1. `npm install`
2. `npm run build`
3. Push to GitHub or use the Vercel CLI to deploy.

I made sure this folder has everything needed for the live site to work perfectly.
