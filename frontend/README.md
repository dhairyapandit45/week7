# Blog App - Frontend
This is the client-side application for the Blog App. I built this using React and Vite to provide a smooth and fast user experience for reading and writing blogs.

## Technologies used
- **Framework**: React (Vite)
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **Routing**: React Router
- **Forms**: React Hook Form
- **HTTP Client**: Axios

## Features
- **Dynamic Routing**: Smooth navigation between home, articles, login, and dashboard.
- **Role-based UI**: Different navigation and features based on whether you are a User, Author, or Admin.
- **Article Feed**: A clean list of articles with profile pictures and category tags.
- **Interactive Forms**: Real-time validation for login, registration, and article creation.
- **Persistent Login**: Uses Zustand and Axios interceptors to keep users logged in.

## Setup and Installation
1. Go to the `frontend` folder.
2. Run `npm install` to install all dependencies.
3. Run `npm run dev` to start the development server locally.
4. For production, run `npm run build`.
